# Excellent-wheel（主要是用于自己学习）

> 带有🤞的表示已经学过了。

本项目主要用来收集轮子类\算法类项目，目前主要收集 Java 和 Go 的，也欢迎其他贡献者提交不同语言实现的轮子类\算法类项目。

轮子类项目可以作为简历上的小项目使用，学习算法类项目可以加深对算法的理解。
